# UseZoracle API Documentation

Welcome to the UseZoracle API! This API allows you to interact with cryptocurrency accounts, check balances, and perform transactions using the Coinbase Developer Platform (CDP) SDK.

## Base URL

```
https://usezoracle-telegrambot-production.up.railway.app
```

## What You Can Do

With the UseZoracle API, you can:

- Create and manage cryptocurrency accounts
- Check token balances for specific accounts
- Transfer tokens between accounts
- Monitor transaction status

## Getting Started

To get started with the UseZoracle API:

1. Review the [Quickstart](/quickstart) guide
2. Set up [Authentication](/authentication)
3. Explore the [API Reference](/api-reference/accounts)
